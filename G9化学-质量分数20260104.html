什么是质量分数？

定义

G9化学中，化合物中某元素的质量分数是指该元素的质量占化合物总质量的百分比。

质量分数是化学计算中的一个重要概念，它帮助我们了解化合物中各元素的含量比例。

计算公式

某元素的质量分数 = (该元素的相对原子质量 × 该元素的原子个数 ÷ 化合物的相对分子质量) × 100%

其中：

  • 某元素的质量分数 = (该元素的相对原子质量 × 该元素的原子个数) ÷ 化合物的相对分子质量 × 100%

常见元素的相对原子质量

元素 符号 相对原子质量 元素 符号 相对原子质量
氢 H 1 碳 C 12
氮 N 14 氧 O 16
钠 Na 23 镁 Mg 24
铝 Al 27 硅 Si 28
磷 P 31 硫 S 32
氯 Cl 35.5 钾 K 39
钙 Ca 40 铁 Fe 56

计算步骤

第一步： 确定化合物的化学式
第二步： 查出各元素的相对原子质量
第三步： 计算化合物的相对分子质量
第四步： 确定目标元素在化合物中的原子个数
第五步： 应用质量分数公式进行计算

例题解析

例题1：计算水（H2O）中氢元素和氧元素的质量分数

解：

第一步： 确定化学式：H2O

第二步： 查相对原子质量：氢(H)的相对原子质量为1，氧(O)的相对原子质量为16

第三步： 计算相对分子质量：水(H2O)的相对分子质量 = 2×氢的相对原子质量 + 氧的相对原子质量 = 2×1 + 16 = 18

氢元素的质量分数：

氢元素的质量分数 = (氢的相对原子质量 × 水分子中氢原子个数 ÷ 水的相对分子质量) × 100% = (1×2 ÷ 18) × 100% = 11.1%

氧元素的质量分数：

氧元素的质量分数 = (氧的相对原子质量 × 水分子中氧原子个数 ÷ 水的相对分子质量) × 100% = (16×1 ÷ 18) × 100% = 88.9%

验证： 11.1% + 88.9% = 100% ✓

例题2：计算碳酸钙（CaCO3）中钙元素的质量分数

解：

第一步： 确定化学式：CaCO3
第二步： 查相对原子质量：钙(Ca)的相对原子质量为40，碳(C)的相对原子质量为12，氧(O)的相对原子质量为16
第三步： 计算相对分子质量：碳酸钙(CaCO3)的相对分子质量 = 钙的相对原子质量 + 碳的相对原子质量 + 3×氧的相对原子质量 = 40 + 12 + 3×16 = 100

钙元素的质量分数：

钙元素的质量分数 = (钙的相对原子质量 × 碳酸钙中钙原子个数 ÷ 碳酸钙的相对分子质量) × 100% = (40×1 ÷ 100) × 100% = 40%

因此，碳酸钙中钙元素的质量分数为40%。

实际应用

质量分数在化学中有广泛应用：

  • 工业生产：计算原料配比，如生产纯碱时需要知道原料中有效成分的含量
  • 环境保护：测定污染物浓度，如空气中二氧化硫的含量
  • 食品科学：检测食品中的营养成分，如蛋白质、脂肪含量
  • 医药领域：药品成分分析，确保药物有效成分的含量符合标准
  • 地质勘探：分析矿石中目标元素的含量，评估矿石价值

练习题

练习1

计算二氧化碳（CO2）中碳元素和氧元素的质量分数。

提示： 碳的相对原子质量为12，氧的相对原子质量为16

练习2

尿素[CO(NH2)2]是常用的氮肥，求尿素中氮元素的质量分数。

提示： 碳的相对原子质量为12，氧的相对原子质量为16，氮的相对原子质量为14，氢的相对原子质量为1

练习3

某化肥主要成分为硝酸铵（NH4NO3），求其中氮元素的质量分数。

提示： 氮的相对原子质量为14，氢的相对原子质量为1，氧的相对原子质量为16

解题技巧

技巧1： 记住常见元素的相对原子质量，特别是H(1)、C(12)、N(14)、O(16)、S(32)、Cl(35.5)、Ca(40)、Fe(56)等。

技巧2： 计算相对分子质量时，注意化学式中各元素的原子个数，不要遗漏下标为1的原子。

技巧3： 计算完成后要进行验证，化合物中所有元素的质量分数之和应等于100%（允许小数点后的误差）。

技巧4： 对于化学方程式的计算，要注意化学方程式的配平，确保反应前后原子个数相等。

技巧5： 对于复杂化学式，可以先分组计算，再求和，如硫酸铜晶体CuSO4·5H2O。

拓展知识

质量分数与质量比的关系：

  • 质量分数是某元素质量占总质量的百分比
  • 质量比是化合物中不同元素质量的比值
  • 例如：水中氢元素与氧元素的质量比为 氢元素质量:氧元素质量 = 2:16 = 1:8

质量分数的用途：

  • 可以用来确定化学式（已知元素质量比时）
  • 可以计算一定质量的化合物中某元素的质量
  • 可以比较不同化合物中同一元素的含量

利用化学方程式的简单计算

在G9化学中，我们经常需要根据化学方程式进行计算，这是化学计算的重要内容之一。化学方程式不仅表示了物质之间的转化关系，还表示了各物质之间的质量关系。

根据化学方程式计算的一般步骤：
1. 设未知数
2. 正确写出化学方程式
3. 计算相关物质的相对分子质量
4. 写出已知量和未知量
5. 列出比例式并求解
6. 简明地写出答案

例题：实验室用碳酸钙(CaCO3)与稀盐酸(HCl)反应制取二氧化碳(CO2)，如果用20g碳酸钙与足量的稀盐酸反应，能够产生多少克二氧化碳？

解：

设： 生成二氧化碳的质量为x

写出化学方程式：
CaCO3 + 2HCl = CaCl2 + H2O + CO2↑

计算相对分子质量：
CaCO3的相对分子质量 = 40 + 12 + 16×3 = 100
CO2的相对分子质量 = 12 + 16×2 = 44

列出已知量和未知量：
100                                                          44
CaCO3 + 2HCl = CaCl2 + H2O + CO2↑
20g                                                                  x

列比例式求解：
100/44 = 20/x
x = (44×20)÷100 = 8.8g

答： 用20g碳酸钙与足量的稀盐酸反应，能够产生8.8克二氧化碳。

练习题

练习1

加热分解31.6g高锰酸钾(KMnO4)，可以得到多少克氧气？（反应方程式：2KMnO4 = K2MnO4 + MnO2 + O2↑）
提示：K=39，Mn=55，O=16

练习2

实验室用锌(Zn)与稀硫酸(H2SO4)反应制取氢气(H2)，化学方程式为：Zn + H2SO4 = ZnSO4 + H2↑。如果用13g锌与足量的稀硫酸反应，能够产生多少克氢气？
提示：Zn=65，H=1

练习3

氢气在氯气(Cl2)中燃烧生成氯化氢(HCl)气体，化学方程式为：H2 + Cl2 = 2HCl。如果燃烧4g氢气，需要消耗多少克氯气？同时生成多少克氯化氢？
提示：H=1，Cl=35.5

练习题解答

练习1解答：

设：生成氧气的质量为x

2KMnO4 = K2MnO4 + MnO2 + O2↑

KMnO4的相对分子质量 = 39 + 55 + 16×4 = 158

O2的相对分子质量 = 16×2 = 32

2×158                                                            32

2KMnO4 = K2MnO4 + MnO2 + O2↑

31.6g                                                                       x

(2×158)/32 = 31.6/x

x = (32×31.6)÷(2×158) = 3.2g

答： 加热分解31.6g高锰酸钾，可以得到3.2克氧气。

练习2解答：

设：生成氢气的质量为x

Zn + H2SO4 = ZnSO4 + H2↑

Zn的相对原子质量 = 65

H2的相对分子质量 = 1×2 = 2

65                                                                      2

Zn + H2SO4 = ZnSO4 + H2↑

13g                                                                        x

65/2 = 13/x

x = (2×13)÷65 = 0.4g

答： 用13g锌与足量的稀硫酸反应，能够产生0.4克氢气。

练习3解答：

设：消耗氯气的质量为x，生成氯化氢的质量为y

H2 + Cl2 = 2HCl

H2的相对分子质量 = 1×2 = 2

Cl2的相对分子质量 = 35.5×2 = 71

HCl的相对分子质量 = 1 + 35.5 = 36.5

2             71                                2×36.5

H2 + Cl2 = 2HCl

4g           x                                    y

2/71 = 4/x

x = (71×4)÷2 = 142g

2/(2×36.5) = 4/y

y = (2×36.5×4)÷2 = 146g

答： 燃烧4g氢气，需要消耗142克氯气，同时生成146克氯化氢。
