G11物理：串并联电路中的功率和做功

课程介绍

本节课我们将学习串并联电路中功率和做功的计算方法，理解电功率与电路连接方式的关系，掌握相关计算公式及其应用。

串并联电路基础知识回顾

1. 串联电路

定义：将电路元件依次连接，接入电路中，电流只有一条通路的电路称为串联电路。

串联电路特点：

  • 电流特点：各处电流相等，即 I = I₁ = I₂ = I₃
  • 电压特点：总电压等于各部分电路电压之和，即 U = U₁ + U₂ + U₃
  • 电阻特点：总电阻等于各部分电阻之和，即 R = R₁ + R₂ + R₃

2. 并联电路

定义：将电路元件并列连接，接入电路中，电流有多条通路的电路称为并联电路。

并联电路特点：

  • 电流特点：总电流等于各支路电流之和，即 I = I₁ + I₂ + I₃
  • 电压特点：各支路两端电压相等，即 U = U₁ = U₂ = U₃
  • 电阻特点：总电阻的倒数等于各支路电阻倒数之和，即 1/R = 1/R₁ + 1/R₂ + 1/R₃

电流、电压、电功的基本定义

1. 电流的定义

电流：表示单位时间内通过导体横截面的电荷量，是描述电流强弱的物理量。

I = Q/t

其中：
I 表示电流，单位为安培(A)
Q 表示电量，单位为库仑(C)
t 表示时间，单位为秒(s)

2. 电压的定义

电压：表示电场力移送单位电荷所做的功，是形成电流的原因。

U = W/Q

其中：
U 表示电压，单位为伏特(V)
W 表示电功，单位为焦耳(J)
Q 表示电量，单位为库仑(C)

3. 电功的定义

电功：表示电流在电路中做功的多少，即电场力移送电荷所做的功。

W = UIt

根据欧姆定律，电功也可表示为：
W = UIt = I²Rt = (U²/R)t

其中：
W 表示电功，单位为焦耳(J)
U 表示电压，单位为伏特(V)
I 表示电流，单位为安培(A)
t 表示时间，单位为秒(s)
R 表示电阻，单位为欧姆(Ω)

4. 串联电路中的电功分配

在串联电路中，各电阻的电流相等，即 I₁ = I₂ = I₃ = I

各电阻在时间t内所做的功为：
W₁ = I²R₁t
W₂ = I²R₂t
W₃ = I²R₃t

总电功为：
W = W₁ + W₂ + W₃ = I²Rt = I²(R₁ + R₂ + R₃)t

电功分配关系：
W₁ : W₂ : W₃ = R₁ : R₂ : R₃

结论：在串联电路中，各电阻在相同时间内做的功与其电阻值成正比，电阻越大，做的功越多。

5. 并联电路中的电功分配

在并联电路中，各电阻两端的电压相等，即 U₁ = U₂ = U₃ = U

各电阻在时间t内所做的功为：
W₁ = (U²/R₁)t
W₂ = (U²/R₂)t
W₃ = (U²/R₃)t

总电功为：
W = W₁ + W₂ + W₃ = (U²/R)t = [U²(1/R₁ + 1/R₂ + 1/R₃)]t

电功分配关系：
W₁ : W₂ : W₃ = 1/R₁ : 1/R₂ : 1/R₃

结论：在并联电路中，各电阻在相同时间内做的功与其电阻值成反比，电阻越大，做的功越少。

6. 电功与电功率的关系

电功与电功率的关系式为：
W = Pt

即：电功 = 电功率 × 时间

电功率的计算公式和原理

1. 电功率的定义

电功率：表示电流做功快慢的物理量，等于单位时间内电流所做的功。

P = W/t

其中：
P 表示电功率，单位为瓦特(W)
W 表示电功，单位为焦耳(J)
t 表示时间，单位为秒(s)

2. 电功率的基本公式

P = UI

根据欧姆定律 U = IR 或 I = U/R，可得电功率的其他表达式：

P = UI = I²R = U²/R

3. 串联电路中的功率分配

在串联电路中，各电阻的电流相等，即 I₁ = I₂ = I₃ = I

各电阻的功率为：
P₁ = I²R₁
P₂ = I²R₂
P₃ = I²R₃

总功率为：
P = P₁ + P₂ + P₃ = I²R = I²(R₁ + R₂ + R₃)

功率分配关系：
P₁ : P₂ : P₃ = R₁ : R₂ : R₃

结论：在串联电路中，各电阻消耗的功率与其电阻值成正比，电阻越大，消耗的功率越大。

4. 并联电路中的功率分配

在并联电路中，各电阻两端的电压相等，即 U₁ = U₂ = U₃ = U

各电阻的功率为：
P₁ = U²/R₁
P₂ = U²/R₂
P₃ = U²/R₃

总功率为：
P = P₁ + P₂ + P₃ = U²/R = U²(1/R₁ + 1/R₂ + 1/R₃)

功率分配关系：
P₁ : P₂ : P₃ = 1/R₁ : 1/R₂ : 1/R₃

结论：在并联电路中，各电阻消耗的功率与其电阻值成反比，电阻越大，消耗的功率越小。

典型例题解析

例题1：串联电路中的功率计算

题目：两个电阻R₁=4Ω，R₂=6Ω串联在电路中，电源电压U=10V，求：

(1) 电路中的电流I

(2) 各电阻消耗的功率P₁、P₂

(3) 电路的总功率P

解答：

(1) 串联电路中总电阻：R = R₁ + R₂ = 4Ω + 6Ω = 10Ω

电路中的电流：I = U/R = 10V/10Ω = 1A

(2) R₁消耗的功率：P₁ = I²R₁ = (1A)² × 4Ω = 4W

R₂消耗的功率：P₂ = I²R₂ = (1A)² × 6Ω = 6W

(3) 电路的总功率：P = UI = 10V × 1A = 10W

或：P = P₁ + P₂ = 4W + 6W = 10W

验证：功率分配比 P₁ : P₂ = 4W : 6W = 2 : 3，电阻比 R₁ : R₂ = 4Ω : 6Ω = 2 : 3，符合 P₁ : P₂ = R₁ : R₂ 的关系。

例题2：并联电路中的功率计算

题目：两个电阻R₁=12Ω，R₂=6Ω并联在电路中，电源电压U=12V，求：

(1) 各支路电流I₁、I₂

(2) 各电阻消耗的功率P₁、P₂

(3) 电路的总功率P

解答：

(1) 并联电路中各支路电压相等

通过R₁的电流：I₁ = U/R₁ = 12V/12Ω = 1A

通过R₂的电流：I₂ = U/R₂ = 12V/6Ω = 2A

(2) R₁消耗的功率：P₁ = U²/R₁ = (12V)²/12Ω = 144/12W = 12W

R₂消耗的功率：P₂ = U²/R₂ = (12V)²/6Ω = 144/6W = 24W

(3) 电路的总功率：P = P₁ + P₂ = 12W + 24W = 36W

或：P = UI = 12V × (1A+2A) = 12V × 3A = 36W

验证：功率分配比 P₁ : P₂ = 12W : 24W = 1 : 2，电阻倒数比 1/R₁ : 1/R₂ = 1/12 : 1/6 = 1 : 2，符合 P₁ : P₂ = 1/R₁ : 1/R₂ 的关系。

例题3：电功的计算

题目：一个电炉的电阻是48.4Ω，接到220V的电路中，通电5分钟，求：

(1) 电炉的功率

(2) 电流做的功

(3) 这些功相当于多少千瓦时(kW·h)

解答：

(1) 电炉的功率：P = U²/R = (220V)²/48.4Ω = 48400/48.4W = 1000W = 1kW

(2) 通电时间：t = 5分钟 = 300秒

电流做的功：W = Pt = 1000W × 300s = 300000J = 3×10⁵J

(3) 1kW·h = 3.6×10⁶J

W = 3×10⁵J = (3×10⁵)/(3.6×10⁶)kW·h = 1/12 kW·h ≈ 0.083kW·h

课堂练习

练习题1

两个电阻R₁=10Ω，R₂=20Ω串联在电路中，电路总电压为6V，求：

(1) 电路中的电流

(2) 各电阻两端的电压

(3) 各电阻消耗的功率

(4) 1分钟内各电阻消耗的电能

答案：

(1) 串联电路中总电阻：R = R₁ + R₂ = 10Ω + 20Ω = 30Ω

电路中的电流：I = U/R = 6V/30Ω = 0.2A

(2) 各电阻两端的电压：

U₁ = IR₁ = 0.2A × 10Ω = 2V

U₂ = IR₂ = 0.2A × 20Ω = 4V

(3) 各电阻消耗的功率：

P₁ = I²R₁ = (0.2A)² × 10Ω = 0.4W

P₂ = I²R₂ = (0.2A)² × 20Ω = 0.8W

(4) 1分钟内各电阻消耗的电能：

时间t = 1分钟 = 60秒

W₁ = I²R₁t = (0.2A)² × 10Ω × 60s = 24J

W₂ = I²R₂t = (0.2A)² × 20Ω × 60s = 48J

练习题2

两个电阻R₁=15Ω，R₂=30Ω并联在电路中，电路总电压为9V，求：

(1) 各支路的电流

(2) 电路的总电流

(3) 各电阻消耗的功率

(4) 2分钟内电路消耗的总电能

答案：

(1) 并联电路中各支路电压相等

通过R₁的电流：I₁ = U/R₁ = 9V/15Ω = 0.6A

通过R₂的电流：I₂ = U/R₂ = 9V/30Ω = 0.3A

(2) 电路总电流：I = I₁ + I₂ = 0.6A + 0.3A = 0.9A

(3) 各电阻消耗的功率：

P₁ = U²/R₁ = (9V)²/15Ω = 81/15W = 5.4W

P₂ = U²/R₂ = (9V)²/30Ω = 81/30W = 2.7W

(4) 2分钟内电路消耗的总电能：

时间t = 2分钟 = 120秒

方法一：W = W₁ + W₂ = P₁t + P₂t = (5.4W + 2.7W) × 120s = 8.1W × 120s = 972J

方法二：总功率 P = P₁ + P₂ = 5.4W + 2.7W = 8.1W

总电能 W = Pt = 8.1W × 120s = 972J

练习题3

R₁=6Ω，R₂=12Ω，R₃=4Ω，电源电压U=12V。

R₂和R₃并联后再与R₁串联，求：

(1) R₂和R₃并联的总电阻

(2) 电路的总电阻

(3) 电路中的总电流

(4) R₁、R₂、R₃各自消耗的功率

(5) 电路的总功率

答案：

(1) R₂和R₃并联的总电阻：

1/R₂₃ = 1/R₂ + 1/R₃ = 1/12Ω + 1/4Ω = 1/12 + 3/12 = 4/12 = 1/3

所以 R₂₃ = 3Ω

(2) 电路的总电阻：

R₁与R₂₃串联，所以 R = R₁ + R₂₃ = 6Ω + 3Ω = 9Ω

(3) 电路中的总电流：

I = U/R = 12V/9Ω = 4/3 A ≈ 1.33A

(4) R₁、R₂、R₃各自消耗的功率：

R₁两端电压：U₁ = IR₁ = (4/3)A × 6Ω = 8V

R₂₃并联部分电压：U₂₃ = U - U₁ = 12V - 8V = 4V

R₁功率：P₁ = U₁²/R₁ = (8V)²/6Ω = 64/6W ≈ 10.67W

或 P₁ = I²R₁ = [(4/3)A]² × 6Ω = 16/9 × 6W = 32/3W ≈ 10.67W

R₂功率：P₂ = U₂₃²/R₂ = (4V)²/12Ω = 16/12W = 4/3W ≈ 1.33W

R₃功率：P₃ = U₂₃²/R₃ = (4V)²/4Ω = 16/4W = 4W

(5) 电路的总功率：

方法一：P = P₁ + P₂ + P₃ = 32/3W + 4/3W + 4W = 36/3W + 4W = 12W + 4W = 16W

方法二：P = UI = 12V × (4/3)A = 16W

课堂小结

1. 电功率是表示电流做功快慢的物理量，计算公式为 P = UI = I²R = U²/R

2. 电功是表示电流做功多少的物理量，计算公式为 W = UIt = I²Rt = (U²/R)t

3. 在串联电路中，功率和电功都与电阻成正比：P₁ : P₂ = R₁ : R₂，W₁ : W₂ = R₁ : R₂

4. 在并联电路中，功率和电功都与电阻成反比：P₁ : P₂ = 1/R₁ : 1/R₂，W₁ : W₂ = 1/R₁ : 1/R₂

5. 无论是串联还是并联电路，总功率等于各部分功率之和：P = P₁ + P₂ + ...

6. 无论是串联还是并联电路，总电功等于各部分电功之和：W = W₁ + W₂ + ...
