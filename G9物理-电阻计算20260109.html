物理习题 - 电路计算

第一页

第1题
已知条件：电源电压 U = 10V，电阻 R = 5Ω

电路类型：简单串联电路（电源 + 开关 + 电阻 R）

  • 求流过电阻的电流是多少安培？
答案：

根据欧姆定律：I = U / R = 10V / 5Ω = 2A

流过电阻的电流是 2 安培。

第2题
已知条件：电源电压 U = 30V，R₁ = 10Ω，R₂ = 20Ω
并联电路
  • (1) 流过 R₁、R₂ 的电流分别是多少安培？
  • (2) 干路电流是多少安培？
  • (3) 电路的总电阻是多少欧姆？
答案：

(1) I₁ = U / R₁ = 30V / 10Ω = 3A

I₂ = U / R₂ = 30V / 20Ω = 1.5A

(2) 干路电流 I = I₁ + I₂ = 3A + 1.5A = 4.5A

(3) 总电阻 R = U / I = 30V / 4.5A = 6.67Ω

或使用并联公式：1/R = 1/R₁ + 1/R₂ = 1/10 + 1/20 = 3/20，所以 R = 20/3 ≈ 6.67Ω

第3题
已知条件：电源电压 U = 50V，L₁、L₂ 电阻分别为 10Ω、40Ω
串联电路（电压表并联在 L₂ 两端）
  • (1) 电路的总电阻？
  • (2) 电路中的电流是多少？
  • (3) 电压表的示数是多少？
答案：

(1) 总电阻 R = R₁ + R₂ = 10Ω + 40Ω = 50Ω

(2) 电路中的电流 I = U / R = 50V / 50Ω = 1A

(3) 电压表的示数（L₂两端的电压）

U₂ = I × R₂ = 1A × 40Ω = 40V

第二页

第4题
已知条件：U = 120V，R₁ = 20Ω，R₂ = 20Ω，R₃ = 50Ω
混联电路（R₁与R₂并联后，再与R₃串联）
  • R₁、R₂并联部分的电阻是多少？
  • 电路的总电阻是多少？
  • 干路中的电流是多少？
  • 流过R₁的电流是多少？
答案：

R₁、R₂并联部分的电阻：

1/R₁₂ = 1/R₁ + 1/R₂ = 1/20 + 1/20 = 2/20 = 1/10

所以 R₁₂ = 10Ω

电路的总电阻：

R = R₁₂ + R₃ = 10Ω + 50Ω = 60Ω

干路中的电流：

I = U / R = 120V / 60Ω = 2A

流过R₁的电流：

I₁ = U₁ / R₁ = (I × R₁₂) / R₁ = (2A × 10Ω) / 20Ω = 1A

或：由于 R₁ = R₂ = 20Ω，并联后电流均分，所以 I₁ = I / 2 = 2A / 2 = 1A

第5题
已知条件：I-U特性曲线图（电流-电压关系图）
I-U特性分析

三条直线A、B、C代表三个电阻的I-U特性：

  • A线：过点(20V, 4A)
  • B线：过点(40V, 4A)
  • C线：过点(40V, 2A)

问题：

  • (1) R_A、R_B、R_C电阻分别是多少？
  • (2) 三个电阻串联后的总电阻是多少？
  • (3) 三个电阻并联后的总电阻是多少？
答案：

(1) 各电阻值：

根据欧姆定律 R = U / I

R_A = 20V / 4A = 5Ω

R_B = 40V / 4A = 10Ω

R_C = 40V / 2A = 20Ω

(2) 串联后的总电阻：

R_串 = R_A + R_B + R_C = 5Ω + 10Ω + 20Ω = 35Ω

(3) 并联后的总电阻：

1/R_并 = 1/R_A + 1/R_B + 1/R_C = 1/5 + 1/10 + 1/20 = 4/20 + 2/20 + 1/20 = 7/20

R_并 = 20/7 ≈ 2.86Ω
