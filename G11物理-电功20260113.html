深入理解电路中的能量转换与功率计算

一、电动势（Electromotive Force, EMF）

1. 定义

电动势是电源将其他形式的能量转换为电能的本领，用符号 E 表示。它反映了电源内部非静电力做功的能力。

E = W非 / q
其中：W非 为非静电力做的功，q 为移动的电荷量

2. 单位

电动势的单位是伏特（V），与电压单位相同。

3. 物理意义

  • 电动势反映电源将其他形式的能量转化为电能的能力
  • 电动势越大，电源将其他能量转化为电能的能力越强
  • 电动势由电源本身的性质决定，与外电路无关
  • 在数值上等于电源未接入电路时两极间的电压
注意：电动势不是力，而是能量转换本领的量度，单位是伏特而非牛顿。

4. 电动势与电压的区别

比较项目 电动势 E 电压 U
物理意义 反映电源将其他能量转化为电能的本领 反映电场力做功的本领
决定因素 由电源本身决定 由电路和电源共同决定
能量转换 其他能量 → 电能 电能 → 其他能量
方向 从电源负极指向正极 从高电势指向低电势

二、电功（Electric Work）

1. 定义

电功是指电场力对电荷所做的功，即电流通过电路时，电能转化为其他形式能量的过程。

W = UIt
其中：U 为电压，I 为电流，t 为通电时间

2. 其他表达形式

W = I²Rt = (U²/R)t
适用于纯电阻电路

3. 单位

电功的国际单位是焦耳（J），常用单位还有千瓦时（kW·h），俗称"度"。

1 kW·h = 3.6 × 10⁶ J

4. 电功的实质

  • 电功是电能转化为其他形式能量的量度
  • 在纯电阻电路中，电能全部转化为内能
  • 在非纯电阻电路中，电能转化为内能和其他形式能量

例题：

一个电阻为 10Ω 的电阻器，两端电压为 220V，通电 10 分钟，求电功是多少？

解：

I = U/R = 220/10 = 22 A

t = 10 min = 600 s

W = UIt = 220 × 22 × 600 = 2,904,000 J = 2.904 × 10⁶ J

三、电功率（Electric Power）

1. 定义

电功率是表示电流做功快慢的物理量，即单位时间内电流所做的功。

P = W/t = UI
其中：W 为电功，t 为时间，U 为电压，I 为电流

2. 其他表达形式

P = I²R = U²/R
适用于纯电阻电路

3. 单位

电功率的国际单位是瓦特（W），常用单位还有千瓦（kW）。

1 kW = 1000 W

4. 额定功率与实际功率

比较项目 额定功率 实际功率
定义 用电器在额定电压下的功率 用电器在实际电压下的功率
特点 由用电器本身决定，固定不变 随实际电压变化而变化
关系 当实际电压等于额定电压时，实际功率等于额定功率
重要：用电器只有在额定电压下才能正常工作，实际功率过大或过小都会影响用电器的寿命或性能。

例题：

一个标有"220V 100W"的灯泡，接在 110V 的电路中，实际功率是多少？

解：

灯泡的电阻：R = U²/P = 220²/100 = 484 Ω

实际功率：P实 = U实²/R = 110²/484 = 25 W

结论：实际功率是额定功率的 1/4

四、焦耳定律（Joule's Law）

1. 内容

电流通过导体产生的热量，与电流的平方成正比，与导体的电阻成正比，与通电时间成正比。

Q = I²Rt
其中：Q 为产生的热量，I 为电流，R 为电阻，t 为时间

2. 适用范围

  • 适用于所有电路（纯电阻和非纯电阻电路）
  • 计算的是电流通过电阻产生的内能

3. 电功与电热的关系

电路类型 电功 W 电热 Q 关系
纯电阻电路 W = UIt Q = I²Rt W = Q（电能全部转化为内能）
非纯电阻电路 W = UIt Q = I²Rt W > Q（电能转化为内能和其他能量）
注意：在非纯电阻电路（如电动机、电解槽等）中，电能 = 内能 + 其他形式的能量，即 W = Q + E其他。

4. 电动与电热的区别

  • 电功（W）：表示电路中消耗的电能，公式为 W = UIt
  • 电热（Q）：表示电流通过电阻产生的热量，公式为 Q = I²Rt
  • 在纯电阻电路中：W = Q
  • 在非纯电阻电路中：W > Q

五、闭合电路欧姆定律

1. 内容

闭合电路中的电流与电源的电动势成正比，与电路的内、外电阻之和成反比。

I = E / (R + r)
其中：E 为电动势，R 为外电阻，r 为内电阻

2. 路端电压

外电路两端的电压称为路端电压，用 U 表示。

U = E - Ir = IR

3. 路端电压与外电阻的关系

  • 当外电阻 R 增大时，电流 I 减小，路端电压 U 增大
  • 当外电路断路时（R → ∞），I = 0，U = E（路端电压等于电动势）
  • 当外电路短路时（R = 0），I = E/r，U = 0
注意：短路时电流很大，容易损坏电源，应避免短路。

4. 电源的输出功率

电源的输出功率是指外电路消耗的功率。

P出 = UI = EI - I²r
  • 当外电阻 R = r 时，电源输出功率最大
  • 最大输出功率：Pmax = E²/4r

例题：

一个电动势为 6V、内电阻为 0.5Ω 的电源，给一个电阻为 2Ω 的用电器供电，求：

（1）电路中的电流

（2）路端电压

（3）电源的输出功率

解：

（1）I = E/(R + r) = 6/(2 + 0.5) = 2.4 A

（2）U = IR = 2.4 × 2 = 4.8 V

（3）P出 = UI = 4.8 × 2.4 = 11.52 W

六、能量守恒在电路中的应用

1. 能量守恒定律

在闭合电路中，电源提供的电能等于内、外电路消耗的电能之和。

EIt = I²rt + I²Rt
电源提供的电能 = 内电路消耗的电能 + 外电路消耗的电能

2. 功率关系

EI = I²r + I²R
电源的总功率 = 内电路消耗的功率 + 外电路消耗的功率

3. 效率

电源的效率是指输出功率与总功率的比值。

η = P出/P总 × 100% = (IR)/(E) × 100% = R/(R + r) × 100%
  • 当外电阻 R 增大时，效率 η 增大
  • 当 R = r 时，η = 50%
  • 当 R >> r 时，η 接近 100%
注意：电源输出功率最大时（R = r），效率只有 50%，并不是效率最高的时候。

七、知识点总结

1. 核心公式汇总

物理量 符号 公式 单位
电动势 E E = W非/q V
电功 W W = UIt = I²Rt = (U²/R)t J
电功率 P P = W/t = UI = I²R = U²/R W
电热 Q Q = I²Rt J
闭合电路电流 I I = E/(R + r) A
路端电压 U U = E - Ir = IR V
电源效率 η η = R/(R + r) × 100% %

2. 重要结论

  • 电动势由电源本身决定，反映电源将其他能量转化为电能的能力
  • 在纯电阻电路中，电功等于电热；在非纯电阻电路中，电功大于电热
  • 当外电阻等于内电阻时，电源输出功率最大
  • 路端电压随外电阻增大而增大
  • 电源效率随外电阻增大而增大

3. 易错点提醒

  • 不要混淆电动势和电压的概念
  • 注意区分额定功率和实际功率
  • 在非纯电阻电路中，不能使用 P = U²/R 或 P = I²R 计算总功率
  • 电源输出功率最大时，效率不是最高的
  • 计算电热时，一定要使用 Q = I²Rt，这个公式适用于所有电路
