化合价

一、化合价的概念

1. 定义

化合价是元素在形成化合物时表现出来的一种性质，是元素原子之间相互化合的数目。

2. 化合价的本质

  • 化合价是元素在化学反应中得失电子或形成共用电子对的数目
  • 金属元素通常显正价，因为它们容易失去电子
  • 非金属元素通常显负价，因为它们容易得到电子

3. 化合价的特点

  • 化合价有正负之分
  • 单质中元素的化合价为零
  • 化合物中正负化合价的代数和等于零
  • 许多元素有多种化合价

例题1：

说明H₂O中各元素的化合价。

氢通常显+1价，氧通常显-2价
在H₂O中：2×(+1) + (-2) = 0 ✓

三、化合价规律和口诀

1. 化合价规律

  • 化合物中正负化合价的代数和等于零
  • 单质中元素的化合价为零
  • 在化合物中，通常氢显+1价，氧显-2价
  • 金属元素通常显正价，非金属元素通常显负价
  • 单原子离子的化合价等于该离子的电荷数

2. 化合价口诀

常见元素化合价口诀：

一价氢氯钾钠银，二价氧钙钡镁锌，
三铝四硅五价磷，二三铁，二四碳，
二四六硫都齐全，铜汞二价最常见。

原子团化合价口诀：

负一硝酸氢氧根，负二硫酸碳酸根，
负三记住磷酸根，正一价的是铵根。

例题2：

根据化合价口诀，写出Al₂(SO₄)₃中各元素的化合价。

铝显+3价，硫酸根显-2价
设硫酸根中硫的化合价为x：
x + 4×(-2) = -2
x - 8 = -2
x = +6
所以硫显+6价，氧显-2价

四、化合价的应用

1. 判断化学式的正误

例题3：

判断下列化学式的正误，并说明理由。

(1) CaO₂：错误，因为钙显+2价，氧显-2价，正负化合价代数和不为零
(2) NaCl₂：错误，因为钠显+1价，氯显-1价，正负化合价代数和不为零
(3) MgO：正确，因为镁显+2价，氧显-2价，正负化合价代数和为零

2. 根据化合价书写化学式

例题4：

写出氧化铝的化学式。

铝显+3价，氧显-2价
利用十字交叉法：
Al³⁺ O²⁻
↓ ↓
3 2
所以化学式为Al₂O₃

例题5：

写出硫酸钙的化学式。

钙显+2价，硫酸根显-2价
Ca²⁺ SO₄²⁻
↓ ↓
2 2
约分后为1：1
所以化学式为CaSO₄

3. 根据化学式计算未知元素的化合价

例题6：

计算KClO₃中氯元素的化合价。

钾显+1价，氧显-2价，设氯的化合价为x
(+1) + x + 3×(-2) = 0
+1 + x - 6 = 0
x - 5 = 0
x = +5
所以氯显+5价

综合练习

练习1：

写出下列物质的化学式：

(1) 氧化镁 (2) 氢氧化钙 (3) 硝酸银 (4) 碳酸钠

(1) MgO：镁显+2价，氧显-2价
(2) Ca(OH)₂：钙显+2价，氢氧根显-1价
(3) AgNO₃：银显+1价，硝酸根显-1价
(4) Na₂CO₃：钠显+1价，碳酸根显-2价

练习2：

计算下列化合物中指定元素的化合价：

(1) H₂SO₄中硫的化合价 (2) Fe₂O₃中铁的化合价 (3) NH₄NO₃中氮元素的化合价

(1) H₂SO₄中硫的化合价：
2×(+1) + x + 4×(-2) = 0
+2 + x - 8 = 0
x = +6

(2) Fe₂O₃中铁的化合价：
2x + 3×(-2) = 0
2x - 6 = 0
x = +3

(3) NH₄NO₃中氮元素的化合价：
铵根中：x + 4×(+1) = +1，x = -3
硝酸根中：y + 3×(-2) = -1，y = +5
所以氮元素分别显-3价和+5价

练习3：

判断下列化学式的正误，错误的请改正：

(1) NaCl₂ (2) CaO (3) Mg(OH) (4) AlSO₄

(1) NaCl₂：错误，应改为NaCl
(2) CaO：正确
(3) Mg(OH)：错误，应改为Mg(OH)₂
(4) AlSO₄：错误，应改为Al₂(SO₄)₃

练习4：

某化合物化学式为H₂RO₄，已知氢为+1价，氧为-2价，求R元素的化合价。

设R元素的化合价为x：
2×(+1) + x + 4×(-2) = 0
+2 + x - 8 = 0
x - 6 = 0
x = +6
所以R元素显+6价

练习5：

配平下列化学方程式：

(1) Fe + O₂ → Fe₂O₃ (2) Al + O₂ → Al₂O₃

(1) 4Fe + 3O₂ → 2Fe₂O₃
(2) 4Al + 3O₂ → 2Al₂O₃

G9课程小结

  • 化合价：元素在形成化合物时表现出来的性质
  • 化合价规律：化合物中正负化合价的代数和等于零
  • 常见化合价：氢+1，氧-2，碱金属+1，碱土金属+2
  • 原子团化合价：氢氧根-1，硫酸根-2，碳酸根-2，铵根+1
  • 应用：判断化学式正误、书写化学式、计算未知元素化合价
  • 记忆方法：利用化合价口诀帮助记忆
