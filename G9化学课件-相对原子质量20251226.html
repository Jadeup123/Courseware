G9化学课件

相对原子质量、相对分子质量、元素的质量分数

一、相对原子质量

1. 概念

相对原子质量是以碳-12原子质量的1/12作为标准，其他原子的平均原子质量与它相比所得的比值。

相对原子质量 = 原子质量 / (碳-12原子质量 × 1/12)

2. 特点

  • 相对原子质量是一个比值，没有单位
  • 相对原子质量≈质子数 + 中子数
  • 元素的相对原子质量可以在元素周期表中查到

例题1：

一个氧原子的质量是2.657×10⁻²⁶kg，一个碳-12原子的质量是1.993×10⁻²⁶kg，求氧的相对原子质量。

氧的相对原子质量 = 2.657×10⁻²⁶kg ÷ (1.993×10⁻²⁶kg × 1/12) ≈ 16

二、相对分子质量

1. 概念

相对分子质量是化学式中各原子的相对原子质量的总和。

相对分子质量 = 各原子的相对原子质量之和

2. 计算方法

  • 确定化学式中各元素的原子个数
  • 查找各元素的相对原子质量
  • 相对分子质量 = Σ(相对原子质量 × 原子个数)

例题2：

计算H₂O（水）的相对分子质量。

水的相对分子质量 = 2×氢的相对原子质量 + 1×氧的相对原子质量 = 2×1 + 1×16 = 18

例题3：

计算H₂SO₄（硫酸）的相对分子质量。

硫酸的相对分子质量 = 2×氢的相对原子质量 + 1×硫的相对原子质量 + 4×氧的相对原子质量 = 2×1 + 1×32 + 4×16 = 98

例题4：

计算Ca(OH)₂（氢氧化钙）的相对分子质量。

氢氧化钙的相对分子质量 = 1×钙的相对原子质量 + 2×[氧的相对原子质量 + 氢的相对原子质量] = 1×40 + 2×(16 + 1) = 74
注意：化学式中括号外的数字表示括号内整个原子团的个数！

三、质量分数

1. 概念

元素的质量分数是指化合物中某元素的质量与化合物总质量的比值。

元素的质量分数 = (元素的质量 / 化合物的质量) × 100%

2. 计算公式

元素的质量分数 = [相对原子质量 × 原子个数 / 相对分子质量] × 100%

3. 计算步骤

  1. 计算化合物的相对分子质量
  2. 计算所求元素的总相对原子质量
  3. 代入公式计算元素的质量分数

例题5：

计算H₂O中氢元素的质量分数。

水的相对分子质量 = 2×氢的相对原子质量 + 1×氧的相对原子质量 = 2×1 + 1×16 = 18
氢元素的质量分数 = [2×氢的相对原子质量 / 水的相对分子质量] × 100% = (2×1 / 18) × 100% = 11.1%

例题6：

计算NH₄NO₃中氮元素的质量分数。

硝酸铵的相对分子质量 = 2×氮的相对原子质量 + 4×氢的相对原子质量 + 3×氧的相对原子质量 = 2×14 + 4×1 + 3×16 = 80
氮元素的质量分数 = [2×氮的相对原子质量 / 硝酸铵的相对分子质量] × 100% = (2×14 / 80) × 100% = 35%

例题7：

计算CaCO₃中钙元素、碳元素、氧元素的质量分数。

碳酸钙的相对分子质量 = 钙的相对原子质量 + 碳的相对原子质量 + 3×氧的相对原子质量 = 40 + 12 + 3×16 = 100
钙元素的质量分数 = [钙的相对原子质量 / 碳酸钙的相对分子质量] × 100% = (40 / 100) × 100% = 40%
碳元素的质量分数 = [碳的相对原子质量 / 碳酸钙的相对分子质量] × 100% = (12 / 100) × 100% = 12%
氧元素的质量分数 = [3×氧的相对原子质量 / 碳酸钙的相对分子质量] × 100% = (3×16 / 100) × 100% = 48%

4. 元素的质量分数的应用

例题8：

100kg NH₄NO₃中含氮元素多少千克？

氮元素的质量分数 = 35%
氮元素质量 = 100kg × 35% = 35kg

综合练习

练习1：

计算C₂H₅OH（乙醇）的相对分子质量。

乙醇的相对分子质量 = 2×碳的相对原子质量 + 6×氢的相对原子质量 + 1×氧的相对原子质量 = 2×12 + 6×1 + 1×16 = 46

练习2：

计算Fe₂O₃中铁元素的质量分数。

氧化铁的相对分子质量 = 2×铁的相对原子质量 + 3×氧的相对原子质量 = 2×56 + 3×16 = 160
铁元素的质量分数 = [2×铁的相对原子质量 / 氧化铁的相对分子质量] × 100% = (2×56 / 160) × 100% = 70%

练习3：

50kg Ca(OH)₂中含钙元素多少千克？

氢氧化钙的相对分子质量 = 钙的相对原子质量 + 2×[氧的相对原子质量 + 氢的相对原子质量] = 40 + 2×(16 + 1) = 74
钙元素的质量分数 = [钙的相对原子质量 / 氢氧化钙的相对分子质量] × 100% = (40 / 74) × 100% ≈ 54.1%
钙元素质量 = 50kg × 54.1% ≈ 27.05kg

练习4：

某尿素样品[CO(NH₂)₂]中氮元素的质量分数为45%，判断该样品是否为纯净物。

尿素的相对分子质量 = 碳的相对原子质量 + 氧的相对原子质量 + 2×[氮的相对原子质量 + 2×氢的相对原子质量] = 12 + 16 + 2×(14 + 2×1) = 60
纯净物中氮元素的质量分数 = [2×氮的相对原子质量 / 尿素的相对分子质量] × 100% = (2×14 / 60) × 100% ≈ 46.7%
45% < 46.7%，所以该样品不是纯净物

G9课程小结

  • 相对原子质量：以碳-12为标准的比值，无单位
  • 相对分子质量：化学式中各原子相对原子质量之和
  • 元素的质量分数：元素质量占化合物总质量的百分比
  • 计算时要特别注意化学式中各元素的原子个数
  • 括号外的数字表示括号内整个原子团的个数
